La función de este programa es leer los datos que proporciona el acelerómetro de tres ejes de la marca
TinkerKit y mostrarlo a través del monitor serie del IDE de Arduino.

Para leer los datos se utilizan los pines A0 y A1 de Arduino. Dichos datos se encuentran dentro del rango
comprendido entre 0 y 255, por lo que se mapean mediante PWM para después ser mostrados.

las modificaciones realizadas por Tom Igoe y Davide Gomba sobre el original.
